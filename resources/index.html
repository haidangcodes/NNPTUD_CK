Job Portal API

Node.js + Express + MongoDB RESTful API

Authentication

POST/api/auth/register - Register new account
POST/api/auth/login - Login and get JWT token

Profiles (Candidate)

GET/api/profiles/me - Get my profile
PUT/api/profiles/me - Update my profile

Companies

GET/api/companies - List all companies
GET/api/companies/:id - Get company by ID
POST/api/companies/me - Create company profile
PUT/api/companies/me - Update my company
PUT/api/companies/:id/approve - Approve company (Admin)

Categories (Admin)

GET/api/categories - List all categories
POST/api/categories - Create category
PUT/api/categories/:id - Update category
DELETE/api/categories/:id - Delete category

Jobs

GET/api/jobs - List jobs (with pagination)
GET/api/jobs/:id - Get job details
POST/api/jobs - Create job (Recruiter)
PUT/api/jobs/:id - Update job
DELETE/api/jobs/:id - Delete job

Applications

POST/api/applications - Apply to job
GET/api/applications/me - My applications
GET/api/applications/job/:jobId - Applications for a job
PUT/api/applications/:id/status - Update status

Interviews

POST/api/interviews - Schedule interview
GET/api/interviews/me - My interviews
PUT/api/interviews/:id - Update interview
DELETE/api/interviews/:id - Cancel interview

Blogs (Admin)

GET/api/blogs - List blogs
GET/api/blogs/:id - Get blog
POST/api/blogs - Create blog
PUT/api/blogs/:id - Update blog
DELETE/api/blogs/:id - Delete blog

Upload

POST/api/upload/cv - Upload CV (PDF)
POST/api/upload/image - Upload image (JPG/PNG)
Note: Include header Authorization: Bearer <token> for protected routes.
